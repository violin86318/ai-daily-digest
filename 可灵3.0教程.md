> 更新时间：2026-02-15

## 一、平台简介

**可灵AI (Kling AI)** 是快手推出的新一代AI生成工具，支持：
- AI生成视频
- AI生成图片
- 图片生成视频
- 视频续写/延长

**官网地址：**
- 国内：https://kling.kuaishou.com 或 https://app.klingai.com
- 海外：https://klingai.com

---

## 二、功能模式

### 1. 文字生成视频 (Text to Video)
输入文本描述，AI自动生成视频

### 2. 图片生成视频 (Image to Video)
上传静态图片，AI将其转化为动态视频

### 3. 视频续写
对已生成的视频进行延长（通常延长5秒）

### 4. 多主体生成
支持在一个画面中生成多个角色

## 三、提示词技巧

### 基本公式
```
[主体] + [场景] + [动作] + [细节] + [风格] + [运镜]
```

### 提示词示例

**示例1：人物**
```
一位年轻女子站在东京涩谷街头，夜晚灯光闪烁，她转身看向镜头，微风吹动她的长发，穿白色衬衫和牛仔裤，电影质感，暖色调，侧面跟随镜头
```

**示例2：风景**
```
日出时分，金色的阳光洒在海面上，波光粼粼，远处的海鸥飞过，航拍视角，缓慢推进镜头，自然风光，梦幻色调，8K画质
```

**示例3：动物**
```
一只橘猫在阳光下的草地上打滚，蝴蝶飞舞，特写镜头，慢动作，温馨治愈风格，浅景深
```

### 常用关键词

**运镜：**
- pan (横摇)
- tilt (俯仰)
- dolly (推拉)
- tracking shot (跟踪镜头)
- aerial view (航拍)
- close-up (特写)
- wide shot (全景)

**光线：**
- warm light (暖光)
- cold light (冷光)
- soft lighting (柔光)
- backlight (逆光)
- golden hour (黄金时刻)

**风格：**
- cinematic (电影质感)
- anime style (动漫风格)
- realistic (写实)
- fantasy (奇幻)
- sci-fi (科幻)

## 四、参数设置

### 生成参数（通常在界面上调整）
- **时长**：5秒 / 10秒 / 更长
- **分辨率**：720p / 1080p / 4K
- **帧率**：24fps / 30fps / 60fps
- **生成数量**：同时生成多条供选择

### 负面提示词
使用负面提示词排除不想要的元素：
```
distorted face, blurry, low quality, extra limbs, watermark, text, logo
```

## 五、使用流程

1. **注册登录**：访问可灵官网，使用手机号或快手账号登录
2. **选择模式**：文字生成视频 / 图片生成视频
3. **编写提示词**：按照上述公式编写
4. **调整参数**：选择时长、分辨率等
5. **生成等待**：通常需要等待1-5分钟
6. **下载保存**：生成完成后可下载高清视频

## 六、进阶技巧

### 1. 分镜提示词
对于复杂场景，可以分段生成后再拼接：
- 远景建立画面
- 中景展示主体
- 特写突出细节

### 2. 关键词权重
在提示词中突出重要元素：
```
(masterpiece:1.2), (best quality:1.2), [主体描述], [场景描述]
```

### 3. 风格一致性
保持整体风格统一：
- 色调一致
- 时代背景一致
- 画面比例一致

## 七、常见问题

**Q: 生成时间很长怎么办？**
A: 高峰期排队人数多，建议避开热门时段，或选择"极速模式"

**Q: 视频模糊怎么处理？**
A: 选择更高分辨率，生成后使用超分工具增强

**Q: 提示词不被识别怎么办？**
A: 使用更简单直接的描述，避免复杂长句
