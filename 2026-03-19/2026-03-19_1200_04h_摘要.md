# 2026-03-19 12:00 AI 摘要

> 核心事件：**NVIDIA 与 AWS 达成重磅合作部署超 100 万 GPU，推理成本骤降将引爆"常驻型"AI Agent 爆发**

---

## 今日焦点

### 1. NVIDIA × AWS：史上最大 AI 基础设施合作
GTC 2026 迎来重磅官宣：
- **NVIDIA 与 AWS 扩大合作伙伴关系**
- AWS 将部署 **超过 100 万块 NVIDIA GPU**
- 涵盖 Blackwell、Rubin GPU 架构及 **Groq 3 LPUs**
- 覆盖 LPUs（语言处理单元）和企业级 RTX PRO Blackwell Server Edition GPUs

### 2. 推理成本骤降 10 倍：AI Agent 经济拐点
Rubin GPU 带来的成本变革：
- **推理成本下降 10 倍** 使之前太贵的用例变得可行
- 将引爆 **"常驻型"（Always-on）AI Agents** 爆发
- 这些 Agent 将持续监控、分析并行动，而非按需调用

### 3. Vera Rubin NVL72 即将登陆 Azure
- **Vera Rubin NVL72 系统准备在 Azure 上推出**
- Azure Local 平台初步支持该架构
- Foundry Agent Service 现已全面可用

### 4. 医疗与生命科学：Agentic AI 拐点已至
- **Roche 宣布部署超过 3,500 块 NVIDIA Blackwell GPUs**
- 覆盖美国和欧洲的混合云及本地环境
- 加速 R&D 生产力、下一代诊断和治疗开发

### 5. NemoClaw 发布：AI Agent 安全补完
- NVIDIA 在 GTC 2026 发布 **NemoClaw**
- 填补 Agent 安全领域空白
- 与 Meta $27B Nebius 交易形成生态呼应
