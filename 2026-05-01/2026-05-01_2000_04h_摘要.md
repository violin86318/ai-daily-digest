# AI 4小时动态摘要 2026-05-01 20:00

Anthropic 推出 Claude Code 和 Managed Agents，OpenAI 发布 GPT-5.5-Cyber

- Anthropic 发布 Claude Code 代理编码系统，支持从审批每步到完全自主模式
- Anthropic Managed Agents 公测，支持多代理协作
- OpenAI Codex 更新支持 GPT-5.5 和 gpt-image-2

## 行业影响
AI 代理进入实用阶段，主要厂商从模型竞争转向代理系统生态构建

## 详细报道
### Anthropic Claude Code 发布
【Anthropic 官网 5月1日报道】
Claude Code 是 Anthropic 推出的代理编码系统，允许开发者控制自主程度：从批准每一步操作，到让内置分类器自动区分安全与风险操作。默认模式谨慎，会在修改文件或执行命令前请求许可。

### Anthropic Managed Agents 公测
【Claude 平台文档 5月1日报道】
Managed Agents 目前处于 beta 阶段，支持多代理协作：一个主代理协调工作、分配子任务、合并结果。Agent SDK 允许构建自定义工作流。

### OpenAI Codex 更新
【Releasebot 5月1日报道】
OpenAI 更新了捆绑的 OpenAI Docs skill，支持 GPT-5.5、gpt-image-2，并提供更清晰的升级指导。同时更新了 contributor 文档和 Rust async trait 指导。

### Anthropic 估值
【Wikipedia 5月1日更新】
Anthropic 截至 2026 年 2 月估值约 3800 亿美元。
