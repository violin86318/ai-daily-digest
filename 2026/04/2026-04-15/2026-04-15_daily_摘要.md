## 3. Claude Code 推出 Routines：让 AI 代理自动化运行

**来源**: Claude Code Docs
**链接**: https://code.claude.com/docs/en/routines

Anthropic 发布 Claude Code **Routines**（研究预览版），允许用户将 Claude Code 配置保存为自动化工作流，通过三种触发方式在云端无人值守运行：

| 触发类型 | 说明 |
|---------|------|
| **Scheduled** | 按计划周期运行（每小时/每天/工作日/每周） |
| **API** | 通过 HTTP POST 到端点触发，支持报警系统、部署流水线集成 |
| **GitHub** | 响应 PR opened/push/issues 等仓库事件自动运行 |

**典型用例：**
- ** backlog 维护**：定时扫描 issue tracker，自动打标签、分配负责人并推送到 Slack
- **告警分类**：监控工具触发 routine → 自动拉取错误日志 → 对比最新提交 → 打开修复 PR
- **代码审查**：PR 打开时自动应用团队审查清单并添加内联评论
- **文档漂移检测**：每周扫描合并的 PR，自动检测过时文档并发起更新 PR
- **SDK 移植**：一个 SDK 仓库合并 PR 后，自动向另一个语言 SDK 发起对应 PR

Routines 运行在 Anthropic 托管的云基础设施上，无需本地设备在线。支持 Pro/Max/Team/Enterprise 计划。
