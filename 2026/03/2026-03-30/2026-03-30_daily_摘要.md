### 1. Claude Code 被曝光：每 10 分钟强制 `git reset --hard origin/main`

> [GitHub Issue #40710](https://github.com/anthropics/claude-code/issues/40710)

Anthropic 的 Claude Code 被用户发现存在严重 bug：每隔 10 分钟自动执行 `git fetch origin + git reset --hard origin/main`，**静默销毁所有未提交的修改**。该操作通过 libgit2 程序化实现，无外部 git 进程，通过 reflog 确认间隔精确为 600 秒。Worktree 不受影响。相关 issue 还有 #8072 和 #7232。这是一个极其危险的行为——对于长时间 coding session 的开发者来说，未保存的工作会不断消失，且 bug 在频繁 commit 的情况下完全不可见。

### 5. ChatGPT 的 Turnstile 验证机制被逆向：55 项指纹检测覆盖浏览器+网络+React 状态

> [buchodi.com](https://www.buchodi.com/chatgpt-wont-let-you-type-until-cloudflare-reads-your-react-state-i-decrypted-the-program-that-does-it/)

作者解密了 ChatGPT 每次发消息时 Cloudflare Turnstile 的验证程序。关键发现：

- 验证程序包含双重加密：外层 XOR key 来自请求本身，内层 key（float 97.35）直接嵌入字节码
- 检查 55 项属性分三层：**浏览器指纹**（WebGL、屏幕、硬件、字体测量）、**Cloudflare 网络层**（边缘注入的 IP/城市/经纬度）、**React 应用状态**（`__reactRouterContext`、`loaderData`、`clientBootstrap`）
- 最关键：验证的不只是"真实浏览器"，而是"已完全启动特定 React 应用的真实浏览器"——一个伪造浏览器指纹但不渲染 ChatGPT SPA 的 bot 会失败
