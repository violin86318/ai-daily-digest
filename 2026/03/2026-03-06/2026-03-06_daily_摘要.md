### 3. Clinejection攻击：GitHub Issue标题如何导致4000台开发者机器被入侵
**来源**: Grist Security | **发布时间**: 2026-03-05

**攻击链**:
1. **Prompt注入**: 攻击者在GitHub Issue标题中嵌入恶意指令
2. **AI执行**: Cline的AI triage bot将注入内容当作指令执行
3. **缓存投毒**: 使用Cacheract工具污染GitHub Actions缓存
4. **凭证窃取**: 窃取NPM_RELEASE_TOKEN、VSCE_PAT、OVSX_PAT
5. **恶意发布**: 发布包含`postinstall`钩子的恶意版本

**影响**:
- 约4000台开发者机器在8小时内安装了OpenClaw
- 攻击利用了npm postinstall钩子的信任链漏洞
- 安全研究员Adnan Khan曾于2026年1月1日报告此漏洞，但未获回应

**防御建议**:
- 消除凭证处理工作流中的GitHub Actions缓存使用
- 采用OIDC provenance attestation进行npm发布
- 建立正式的漏洞披露流程

**链接**: [A GitHub Issue Title Compromised 4,000 Developer Machines](https://grith.ai/blog/clinejection-when-your-ai-tool-installs-another)
