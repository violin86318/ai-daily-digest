## 2. 智能体时代重新审视文学编程

## 5. Rust Wasm 编写实践笔记

**核心建议（TL;DR）**：
1. 所有跨边界传递使用 `&reference`
2. 用 `Rc<RefCell<T>>` 或 `Arc<Mutex<T>>` 代替 `&mut`
3. 不要对导出类型派生 `Copy`
4. 跨边界的集合类型使用 `wasm_refgen`
5. 导出的 Rust 类型加 `Wasm*` 前缀，导入的 JS 类型加 `Js*` 前缀

**价值点**：作者多年踩坑总结的实战模式，非常实用。

🔗 https://notes.brooklynzelenka.com/Blog/Notes-on-Writing-Wasm
