# AI Weekly Digest Image Prompt

Generate a 3:4 vertical information card (1080x1440px) summarizing the most important AI/tech news from the week of March 2-9, 2026.

## Visual Style
- Modern, clean tech journalism aesthetic
- Dark theme with vibrant accent colors
- Use icons or small illustrations for each news item
- Font: Bold sans-serif for headlines, clean body text

## Layout (Top to Bottom)

### Module 1: Header
- Title: "AI WEEKLY DIGEST"
- Subtitle: "March 2-9, 2026"
- Decorative line/element

## Color Palette
- Background: Deep dark blue/black (#0a0a1a)
- Primary text: White (#ffffff)
- Accent colors: 
  - Gold/Yellow for money (#ffd700)
  - Blue for tech (#4a9eff)
  - Red for alerts (#ff4757)
  - Green for positive (#2ed573)
- Secondary text: Light gray (#a0a0b0)

## Style Notes
- Each module should be visually distinct (slight background variation or border)
- Maintain good whitespace between sections
- Use consistent icon sizing
- Make it scrollable-friendly or single-screen readable
