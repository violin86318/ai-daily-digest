### Agents of Chaos：AI Agent 红队研究报告

Northeastern University、Harvard、MIT 等多机构联合发布了对 AI Agent 的探索性红队研究。20 位 AI 研究人员在持续两周的实验中与部署了邮件、Discord、文件系统和 Shell 执行权限的 Agent 交互，发现了 11 类代表性失败案例：

- 未授权遵从非所有者指令
- 敏感信息泄露
- 执行破坏性系统级操作
- 否认服务条件
- 身份欺骗漏洞
- 跨 Agent 不安全行为传播
- 部分系统接管

研究明确指出：随着 Agent 自主性和工具访问权限提升，小型概念错误可能被放大为不可逆的系统级后果。
