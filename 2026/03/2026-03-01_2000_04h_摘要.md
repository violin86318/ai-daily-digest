## 🔬 AI 研究与教育
### microgpt：极简 GPT 的美学
Karpathy 发布 microgpt，仅用 200 行纯 Python（无依赖）实现完整 GPT 训练与推理。包含数据集处理、tokenizer、autograd 引擎、GPT-2 架构、Adam 优化器、训练与推理循环。训练 1000 步后 loss 从 3.3 降至 2.37，可生成新的名字如 "karia"、"yerna"。项目体现了 Karpathy 十年间将 LLM 简化到极致的追求。

**来源：** [Karpathy Blog](http://karpathy.github.io/2026/02/12/microgpt/)

## 🛠️ 工程与工具
### Python Monorepo 最佳实践
文章介绍使用 uv + Dagger 构建高效 Python monorepo 的方法。uv 的 workspace 概念简化了多包管理，Dagger 管道可用 Python 编写，支持 BuildKit 优化、并行化和端到端缓存（包括构建镜像后的 linter 和测试步骤）。现代 QA 工具 ruff 和 pyright 原生支持 monorepo。

**来源：** [Daniel Gafni Blog](http://gafni.dev/blog/cracking-the-python-monorepo/)

### PostgreSQL 随机 I/O 成本实测
作者通过实验测量 PostgreSQL 的 random_page_cost 真实开销。实验显示：本地 SSD 上随机读开销约为顺序读的 25-35 倍，远超默认值 4.0。这解释了为何 SSD 上降低 random_page_cost 的建议可能适得其反——默认值已被严重低估。

**来源：** [Tomas Vondra Blog](http://vondra.me/posts/the-real-cost-of-random-io/)
