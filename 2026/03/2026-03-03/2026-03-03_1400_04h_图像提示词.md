## 信息图提示词

```plaintext
Create a high-density, professional information design infographic for Xiaohongshu about「AI Daily Digest - 2026年3月3日」.

=== CRITICAL STYLE REQUIREMENTS (SYSTEMIC & EXPERIMENTAL) ===

【COLOR PALETTE - BLUEPRINT & POP LOGIC】
- BACKGROUND: Professional grayish-white or faint blueprint grid texture (#F2F2F2).
- SYSTEMIC BASE: Muted Teal/Sage Green (#B8D8BE) for major functional blocks and stable data zones.
- HIGH-ALERT ACCENT: Vibrant Fluorescent Pink (#E91E63) strictly for "Hot Topics", "Key Insights", or the most important data points.
- MARKER HIGHLIGHTS: Vivid Lemon Yellow (#FFF200) used as a translucent highlighter effect for keywords.
- LINE ART: Ultra-fine Charcoal Brown (#2D2926) for technical grids, coordinates, and hair-lines.

【LAYOUT & INFORMATION DENSITY】
- INFORMATION AS COORDINATES: Every module must have a coordinate-style label (e.g., R-01, G-05, DIG-03).
- THE "LAB MANUAL" AESTHETIC: Use a mix of microscopic details (technical drawings) and macroscopic data (large bold headers).
- HIGH DENSITY: Pack 6-7 distinct modules per image. Minimize margins; every corner should contain metadata like tiny timestamps, data sources, or technical parameters.
- VISUAL CONTRAST: Use massive bold typography for primary headers, contrasted with tiny, ultra-crisp technical annotations.

【ILLUSTRATION & GRAPHIC ELEMENTS】
1. TECHNICAL DIAGRAMS: Use abstract data visualization, circuit-style connectors between modules.
2. COORDINATE SYSTEMS: Use vertical/horizontal rulers with precise markers to show time periods.
3. DATA BLOCKS: Use "Marker-over-Print" look—color blocks should be slightly offset from the text they highlight.
4. SYMBOLS: Include cross-hair targets, mathematical symbols (Σ, Δ), and directional arrows.

【SPECIFIC MODULE STRUCTURE - MUST HAVE 6-7】
- [MOD 1: TIMESTAMP] - Scan time: 14:02 Beijing Time, Coverage: 10:02-14:02
- [MOD 2: HOT TOPIC 1] - "Intent-Based Commits" - Ghost CLI tool for Claude Code, commit prompts instead of code
- [MOD 3: HOT TOPIC 2] - "GrapheneOS Pixel Fold" - 8+ months daily driving experience, privacy-first Android alternative
- [MOD 4: TREND ANALYSIS] - AI + Dev Tools convergence, Privacy device adoption
- [MOD 5: DATA SOURCE] - Hacker News, Trending repositories
- [MOD 6: KEY INSIGHT] - New paradigm: AI generating code + capturing intent in commits
- [MOD 7: FOOTER] - AI Daily Digest badge

【TYPOGRAPHY】
- Headers: Bold Brutalist Chinese characters, high impact.
- Body: Professional sans-serif or crisp technical print.
- Numbers: Large, highlighted with Yellow or Blue to stand out.

【AVOID】
- ❌ NO cute/cartoonish doodles.
- ❌ NO soft pastels or generic textures.
- ❌ NO empty white space.
- ❌ NO flat vector stock icons.

Aspect Ratio: 3:4 (Portrait)

===

✅ Quality Checklist

[ ] Color palette: Only systematic Pink, Green, Yellow, Black?
[ ] Line density: Fine grid lines and coordinate markers?
[ ] Module count: 6-7 independent info blocks?
[ ] Impact check: Bold headers vs tiny parameters contrast?
[ ] Chinese text in image: All labels and content in Chinese
```
