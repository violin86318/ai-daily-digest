## 🔬 Flash-MoE: 397B 模型笔记本上跑 4.4 tok/s

**Flash-MoE** 是一个纯 C/Metal 推理引擎，可在 MacBook Pro（48GB RAM）上运行 Qwen3.5-397B-A17B（3970亿参数 MoE 模型），达到 **4.4 tokens/秒**，完整支持 tool calling。

核心亮点：
- 209GB 模型从 SSD 流式传输，通过 Metal 自定义计算管线
- "Trust the OS" 策略：OS 页缓存自然达到 71% 命中率，自定义 LRU 反而更慢
- FMA 优化去量化内核：12% 性能提升
- 串行流水线（GPU → SSD → GPU）经实测是硬件最优，因 Apple Silicon 统一内存架构下 SSD DMA 和 GPU 计算共享内存控制器，重叠反而更差

**关键教训**：看似应该重叠的 I/O 和计算，在 Apple M3 Max 上反而必须串行。

## 🔐 Charles Bennett 获 Turing 奖：量子密码学的诞生

IBM 科学家 Charles H. Bennett 与 Gilles Brassard 共同获得 2025 年 Turing 奖（计算机界诺贝尔奖），表彰其在**量子密码学**上的开创性贡献。

1979 年两人在 Puerto Rico 海边游泳时偶遇，聊出了 BB84 协议——利用量子力学定律实现理论不可破解的密钥分发。1989 年 Bennett 在 IBM 办公室用镜子、偏振片和光子探测器造出了世界上第一台量子密码机。

Bennett 对量子信息的比喻："量子信息就像梦中的信息。一旦你开始向别人描述你的梦，你就开始忘记梦，只记得你说的话。公开版本可以被复制，但它不等于梦本身。"

## 📦 Manyana：CRDT 重写版本控制的未来

Bram Cohen 发布 Manyana，用 **CRDT（无冲突复制数据类型）** 重构版本控制的核心设计。

传统 VCS 的合并冲突是两个不透明 blob；Manyana 告诉你"左删除了函数，右在中间插入了一行"。合并永远不会失败，冲突是被标记出来的，而非阻塞性的。

核心洞察：Rebase 不必摧毁历史——CRDT 可以保留完整历史同时呈现"提交在最新 main 上"的线性视图。

## 📱 Teaching Claude to QA a Mobile App

独立开发者 Christopher Meiklejohn 分享了用 Claude 自动测试其跨平台应用 Zabriskie 的经历。

**Android：90 分钟搞定**
- 利用 Android WebView 的 Chrome DevTools Protocol socket
- Python 脚本 90 秒扫完 25 个屏幕，自动截图分析并提交 bug 报告

**iOS：6+ 小时噩梦**
- 无法输入 @ 符号（AppleScript 的 Shift+2 被 iOS Simulator 解析为快捷键）
- 无法关闭 UIKit 原生通知对话框（无法通过 macOS 合成输入）
- 无法从 macOS 剪贴板粘贴到 iOS Simulator（两个独立剪贴板系统）

教训：**跨平台开发工具链的碎片化是 2026 年移动开发的最大痛点。**

## 🖥️ Windows 原生开发：一团乱麻

开发者 Domenic 记录了用 WinUI 3 写一个简单显示器黑屏工具的血泪史。

Windows UI 框架演化路径：**Win32 C APIs → MFC → WinForms → WPF → WinRT XAML → UWP XAML → WinUI 3**

## 🧠 代码之死？Steve Krouse 论 AI 与编程的未来

"Reports of code's death are greatly exaggerated"——文章反驳"AI 将消灭代码"的观点。

核心论点：
1. **抽象是人类的极限工具**：大脑只能同时处理 7±2 个概念，抽象让我们压缩无限复杂度。好的抽象是诗，不是模糊。
2. **代码本身是重要 artifact**：不是只为了产出软件，代码本身是智力成果。
3. **vibe coding 会泄露**：当系统足够复杂，低层次抽象的漏洞就会爆发。Dan Shipper 的 vibe-coded app 因"实时协作"崩了——这个功能看起来简单，实际上是噩梦级的。
4. **AGI 会帮助我们造更好的抽象**：不是替代代码，而是帮我们解决最难的多人协作编辑器问题。

## 📊 Simon Willison 的 HN 用户画像实验
