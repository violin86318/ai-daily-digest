## 详细报道

### 1. Claude Opus 4.7 发布 (主要事件)

**发生了什么:** Anthropic 发布 Claude Opus 4.7，这是其迄今为止最强大的公开可用语言模型。

**关键数据:**
- **知识工作 (GDPVal-AA):** Elo 1753，超越 GPT-5.4 (1674) 和 Gemini 3.1 Pro (1314)
- **代理编码 (SWE-bench Pro):** 64.3% 任务解决率 vs Opus 4.6 的 53.4%
- **研究生级推理 (GPQA):** 94.2%
- **视觉推理:** 98.5% (XBOW 基准，从 54.5% 大幅提升)
- **分辨率提升:** 支持至高 2,576 像素 (约 3.75MP)，3倍提升

**新功能:**
- **"effort" 参数:** 新增 xhigh 级别，提供性能和成本的更细粒度控制
- **任务预算 (Task Budgets):** 公开测试版，允许设置 token 花费上限
- **/ultrareview 命令:** Claude Code 中新增的高级代码审查
- **Auto Mode 扩展:** 开放给 Max 计划用户

**定价:** 保持不变，$5/$25 每百万 tokens

**安全:** 包含自动检测和阻止高风险网络安全请求的防护措施

### 2. Claude Mythos Preview 网络安全能力

**发生了什么:** Anthropic 透露其更强大的模型 Claude Mythos Preview 已完成 32 步企业级网络攻击测试，这是首个 AI 系统达成此成就。

**背景:** 该模型因风险过高未公开，仅限少数企业合作伙伴用于网络安全测试。Anthropic 启动 "Project Glasswing" 和 Cyber Verification Program。
