### 1. Anthropic 切断 OpenClaw 第三方工具支持 (重磅)

**事件概述:**
Anthropic 宣布从 2026年4月4日 12:00 PM PT (北京时间 4月5日凌晨3点) 起，Claude 订阅用户将无法再使用第三方 agent 工具如 OpenClaw。

**关键细节:**
- 原因: 第三方工具未优化 prompt cache hit rate，给 Anthropic 造成巨大计算资源压力
- 官方说法: "Capacity is a resource we manage thoughtfully and we are prioritizing our customers using our core products and API"
- OpenClaw 创始人 Peter Steinberger (已被 OpenAI 雇佣) 质疑: "First they copy some popular features into their closed harness, then they lock out open source"

**用户影响:**
- 订阅用户仍可使用 Claude，但需要额外购买 "Extra Usage" 包
- 一次 性 credit：等于月费金额，有效期至 4月17日
- 折扣：预购 extra usage 可享最高 30% 折扣
- 成本估算：单个 OpenClaw agent 运行一天可能消耗 $1,000-$5,000 API 费用

**技术背景:**
- Anthropic 第一方工具 (Claude Code, Claude Cowork) 针对 prompt cache 做了优化
- 第三方工具未做类似优化，导致计算资源浪费
- 近期还施加了更严格的 session limits (每5小时)

### 2. Google 发布 Gemma 4

**发布时间:** 2026年4月2日

**模型规格:**
- 四种规格: Effective 2B (E2B), Effective 4B (E4B), 26B MoE, 31B Dense
- 许可证: Apache 2.0 (从 Llama 3.1 style 切换)

**核心特性:**
- 专为 advanced reasoning 和 agentic workflows 设计
- 原生 function calling 支持
- 结构化 JSON 输出
- 边缘设备优化 (Android 等)
- 超越简单对话，处理复杂逻辑

**市场意义:**
- 首个真正面向本地 agentic AI 的开源模型
- 在参数效率上有突破性进步
